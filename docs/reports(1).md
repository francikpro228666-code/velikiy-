# Отчёты в приложении «Финансовый дневник»

## 1. Отчёт по доходам и расходам за период
Пользователь выбирает дату начала и окончания. Отчёт отображает:
- Общая сумма доходов
- Общая сумма расходов
- Баланс (доходы – расходы)
- Таблица доходов по категориям (сумма по каждой категории)
- Таблица расходов по категориям (сумма по каждой категории)

## 2. Отчёт по бюджету за месяц
Пользователь выбирает месяц. Отчёт показывает для каждой категории расходов, на которую установлен лимит:
- Название категории
- Установленный лимит
- Фактически потраченная сумма
- Остаток (лимит – потрачено)
- Процент использования лимита

## 3. Отчёт по финансовым целям
Отчёт отображает все цели пользователя:
- Название цели
- Целевая сумма
- Накопленная сумма
- Процент выполнения
- Статус (выполнена / в процессе)
- Дедлайн (если указан)

## Выгрузка отчётов
Каждый отчёт можно:
- Посмотреть в виде HTML-таблицы на странице
- Скачать в формате **Microsoft Excel (XLSX)**
- Скачать в формате **Microsoft Word (DOCX)**
