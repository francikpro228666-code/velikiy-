\# Отчёты в приложении «Финансовый дневник»



\## 1. Отчёт по доходам и расходам за период

Пользователь выбирает произвольный интервал дат. Отчёт содержит:

\- Общую сумму доходов за период

\- Общую сумму расходов за период

\- Баланс (доходы минус расходы)

\- Детализацию по категориям: отдельно доходы по категориям, отдельно расходы по категориям с суммами.



\## 2. Отчёт по бюджету за месяц

Пользователь выбирает месяц. Отчёт показывает для каждой категории расходов, на которую установлен лимит:

\- Название категории

\- Установленный лимит

\- Фактически потраченную сумму

\- Остаток (лимит минус потрачено)

\- Процент использования лимита



\## 3. Отчёт по финансовым целям

Отчёт отображает все цели пользователя с полями:

\- Название цели

\- Целевая сумма (нужно)

\- Текущая накопленная сумма

\- Процент выполнения

\- Статус (выполнена / в процессе)

\- Дедлайн (если указан)



\## Форматы выгрузки

Каждый из этих отчётов можно:

\- Просмотреть в виде HTML-таблицы на странице

\- Скачать в формате \*\*Excel (XLSX)\*\*

\- Скачать в формате \*\*Word (DOCX)\*\*
